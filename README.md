# waste-recycling-data-analysis
Progetto di analisi dei dati sulla raccolta differenziata nei comuni dell’Emilia-Romagna. L’analisi utilizza modelli di regressione, metodi di selezione delle variabili (best subset, BIC) e tecniche di resampling (LOOCV, bootstrap) per individuare i fattori che influenzano le performance ambientali.
